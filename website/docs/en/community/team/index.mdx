# Core team

The development of Rspack stack is led by ByteDance's web infra team and driven together with community contributors on several core projects, including [Rspack](https://github.com/web-infra-dev/rspack), [Rsbuild](https://github.com/web-infra-dev/rsbuild), [Rspress](https://github.com/web-infra-dev/rspress), [Rsdoctor](https://github.com/web-infra-dev/rsdoctor) and [Rslib](https://github.com/web-infra-dev/rslib).

Electron-Rspack is an independent project. Current members of the Electron-Rspack team are listed in random order below.
